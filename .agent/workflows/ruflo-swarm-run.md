# Workflow: ruflo Swarm Run

Current status: planning only.

1. Create a dedicated PRD task.
2. Use a branch or temporary copy.
3. Record every generated file.
4. Verify ruflo memory does not replace `context/`.
5. Verify ruflo goals do not replace `prds/json/`.
6. Produce `docs/ruflo-integration-audit.md`.

Only merge after human walkthrough approval.
