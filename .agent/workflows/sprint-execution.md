# Workflow: Sprint Execution

1. Read `Agent.md`.
2. Read the active PRD JSON.
3. Confirm owned files and permission level.
4. Execute local edits.
5. Run verification.
6. Update PRD status.
7. Update context if a material decision changed.
8. Prepare walkthrough notes for human review.

Do not perform external writes unless the task is L4/L5 and explicitly approved.
