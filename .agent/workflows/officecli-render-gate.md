# Workflow: OfficeCLI Render Gate

Use for Word, Excel, PowerPoint, or Office-derived final artifacts.

1. Inspect document structure.
2. Apply scoped edits.
3. Render HTML, PDF, PNG, or screenshot preview.
4. Check overflow, missing fonts, bad tables, broken links, and formatting drift.
5. Save validation evidence.
6. Record residual risks.

Visual artifacts are not done until they are rendered and inspected.
