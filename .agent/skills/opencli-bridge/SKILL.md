# Skill: OpenCLI Bridge

Use when a task needs logged-in browser or platform operations.

Permission:

- External read: L3.
- External write: L4.
- Sensitive account actions: L5 and default deny.

Rules:

- Do not silently publish.
- Do not bypass platform safety checks.
- Prefer filling drafts and asking the human to confirm.
- Write audit records after external writes.
