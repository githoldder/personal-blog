# Public User Manual

Readers can use the site as a public knowledge portfolio:

- Home: current narrative and featured knowledge assets.
- Notes: public Markdown reader pages.
- Library: books, notes and curated resources.
- Projects: portfolio entries with technical and product context.
- Timeline: chronological view across learning, projects and assets.
- Graph Lab: semantic map for exploring public relationships.
- Resume: generated PDF and structured resume facts.

Private Obsidian source files, certificates and local admin operations are not part of the public reader experience.
