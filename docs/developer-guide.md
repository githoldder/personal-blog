# Developer Guide

## Setup

```bash
npm install
npm run dev
```

## Core Commands

- `npm run build:assets`
- `npm run build:resume`
- `npm run build:semantic-intelligence`
- `npm run validate:prd`
- `npm run validate:pipeline`
- `npm test`
- `npm run verify`
- `npm run api:local`

## Conventions

PRD changes live in `prds/json` and `prds/md`. Generated public assets live under `public/assets`. Private evidence remains under `content/resume/references/evidence` and is excluded from public packages unless approved.
